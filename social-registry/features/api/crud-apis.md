# CRUD APIs

The APIs are divided into 3 parts:

1. Individual
2. Group
3. Common APIs

## Individual

1. Get
   1. Get by id
   2. Get by NAME, ID\_TYPE, ID\_VALUE
2. Create

## Group

1. Get
   1. Get by id
   2. Get by NAME, ID\_TYPE, ID\_VALUE
2. Create
   1. Create Group

## Common APIs

1. Update ID
   1. Search for registrant by id
   2. Update id
2. Add relationship
3. Map individual with group
